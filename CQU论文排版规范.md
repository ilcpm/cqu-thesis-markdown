# 重庆大学本科毕业论文排版规范

这里根据本科论文的毕业论文写作要求，抽离了排版要求，（**仅抽离了排版要求**）

规范来源：见[符号定义](#符号定义)章节

## 符号定义

为说明格式规范来源，在对应内容尾部采用括号说明，这里对括号内容进行说明

- 仅数字，如：1.1
  - 来自文件《附件 3 重庆大学普通本科毕业设计（论文）撰写规范要求》
  - 文件修改日期为2017年，2019年沿用
  - 文件内容核心大致采纳自研究生版本的排版要求，可以看出复制粘贴的痕迹
- 带字母E（Example），如：E1.1
  - 来自本科19年排版范例《附件4 重庆大学普通本科毕业设计（论文）模版201904》

## 存在的问题

《附件 3 重庆大学普通本科毕业设计（论文）撰写规范要求》中的若干存疑事项

- 3 和 4 节中提到了“封面”和“中英文题名页”

  - 封面和题名页应该是一个东西才对，但是明确单独同时提到了封面和题名页
  - 从给的排版范例来看，应该只有“中英文题名页”，也就是通俗意义的中文封面和英文封面
  - 没有“封面”这一项目
  - （疑似封面单独打印填写放在最上面装订，不属于写作时候的内容 🤔）

## 论文结构(3.2)

- 中文封面（题名页）
- 英文封面（题名页）
- （分节）
- 中文摘要（含关键词）
- 英文摘要（含关键词）
- 目录
- 插图和附表清单（即图目录，表目录）（非必须，暂不处理）
- 符号、标志、缩略词、首字母缩写、计量、单位、术语等注释表(即符号术语表)（非必须，暂不处理）
- （分节）
- 正文
  - 引言（续论）(5.3)
  - 正文(5.4)
  - 结论与展望(5.5)
  - 致谢(5.6)
  - 参考文献(5.7)
  - 附录(6)

## 样式

### 主题字体

- 中文标题：黑体
- 中文正文：宋体
- 西文标题：Times New Roman
- 西文正文：Times New Roman

### 普通文本

- 正文(2.5)
  - 字体：
    - 字体：正文字体
    - 字号：小四
    - 字体间距：标准
  - 段落：
    - 行距：固定值 20 磅
- TODO 关键词（暂未发现规定）
- 题注(5.4.1)
  - 字体：
    - 字号：五号
    - 字体：正文字体
  - 段落：
    - 居中（未定义，根据惯例处理）
- 脚注：未定义，继承题注样式
- 页眉(2.2)
  - 字体：
    - 字体：正文字体
    - 字号：5
  - 边框：
    - 下分隔线
- 页脚：未定义，继承页眉，去掉下框线

### 标题样式

- 一级标题
  - 字体：(5.2.1)
    - 字号：三号
    - 字体：标题字体
  - 段落：
    - 居中(5.2.1)
    - 段前分页(5.1)
- 二级标题
  - 字体：(5.2.1)
    - 字号：三号
    - 字体：标题字体
  - 段落：
    - 左对齐(5.2.1)
  - TODO 二级标题之间空行???(5.1)
- 三级标题
  - 字体:(5.2.1)
    - 字号：四号
    - 字体：标题字体
  - 段落：
    - 左对齐(5.2.1)
- 四级标题
  - 字体：(5.2.1)
    - 字号：继承正文
    - 字体：继承正文
- 五级标题(5.2.1)
  - 同四级
- 更多未定义，继承正文

### 编号

#### 标题编号(5.2.1)

- 文科（文、法、教育等）

  | 标题层级 | 序号样式 |
  | :------- | :------- |
  | 1        | 一、     |
  | 2        | （一）   |
  | 3        | 1.       |
  | 4        | （1）    |
  | 5        | 第一.    |

- 理工经管

  | 标题层级 | 序号样式 |
  | :------- | :------- |
  | 1        | 1        |
  | 2        | 1.1      |
  | 3        | 1.1.1    |
  | 4        | ①       |
  | 5        | 1）      |
  | 6        | a.       |

#### 题注编号

- 作用域：正文(5.2.2)
  - 图 2.1、表 3.2、式（3.5）
- 作用域：附件(5.2.4)
  - 设计或论文的附录依序用大写正体英文字母A、B、C……编序号，如：附录A
  - 图A1；表B2；式(B3)；文献[A5]等

#### 脚注编号(5.4.4)

带圈字符①②③

## 特殊部分

### 页面纸张(2.1)

- 纸张：A4
- 页边距：
  - 上：30mm
  - 下：25mm
  - 左：25mm
  - 右：25mm
  - 装订线：10mm
    - 装订线会根据奇偶页面，在扣除左右边距之后，再留出相应的宽度作为装订的预留空间
  - 页眉：16mm
  - 页脚：15mm

### 页眉页脚

#### 页眉(2.2)

从摘要开始，摘要之前用分节符分节设置不同的页眉

- 作用域：摘要，目录，正文，参考文献，附录
  - 单面打印：
    - 左对齐：重庆大学本科学生毕业设计（论文）
    - 右对齐：标题 1
  - 双面打印：
    - 奇数页（右页）：
      - 居中：标题 1
    - 偶数页（左页）：
      - 居中：重庆大学本科学生毕业设计（论文）

#### 页脚(2.3, 2.4, 6.2)

从摘要开始，同页眉

- 作用域：摘要，目录
  - 居中：大写罗马数字
- 作用域：正文，参考文献，附录
  - 居中：阿拉伯数字

### 封面(4.2)

分中英文摘要，中间分页，封面完毕需要分页

- 重庆大学本科学生毕业设计（论文）（小二号黑体，居中）
- 设计或论文题名（二号黑体，居中）
- 重庆大学校徽（35×35mm 居中）
- 学生：××××××××（三号黑体，居中）
- 学号：××××××××（中文三号黑体，数字三号 Times New Roman，居中）
- 指导教师：（×××××××× 三号黑体，居中）
- 助理或校外指导教师：（×××××××× 三号黑体，居中）（必要时）
- 专业：××××××××（三号黑体，居中）
- 所在院（系、所）（小二黑体，居中）
- 论文提交（完成）时间（三号黑体，居中）

### 摘要(4.4)

分中英文摘要，中间分页，摘要完毕需要分页

- 禁止插图（根据 5.2.2 未定义摘要的图序号）
- 关键词(4.5)

### 目录(4.6)

- 插图与附表清单(4.7)
  - 分别放置于目录之后
- 符号、标志、缩略词、首字母缩写、计量单位、名词、术语等的注释表(4.8)
  - 图标清单页之后

### 参考文献(5.7)

遵循标准《文后参考文献著录规则》（GB/T 7714-2005）

最新版本为2015

### 附录(5.2.4)

采用`附录A`，`附录B`，`附录C`的方式
